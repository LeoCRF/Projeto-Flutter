# Focus.Me

Focus.Me é um gerenciador de rotina e foco diário, com registro de tarefas, humor e estatísticas.

## Pré-requisitos

Antes de rodar o projeto, você precisa ter instalado:

- [Flutter](https://flutter.dev/docs/get-started/install) (>= 3.0.0 <4.0.0)
- Um editor de código (VS Code ou Android Studio)
- Um dispositivo físico ou emulador para testar o app
- Arquivo do Firebase configurado (`google-services.json` para Android e `GoogleService-Info.plist` para iOS)
